do...while循环

注意while和do...while语句的区别
while是先判断条件，再根据判断结果若为true则执行循环体，否则跳过循环体。
do...while是先执行一次循环体，然后再进行判断，再根据判断结果若为true则继续执行循环体，否则跳过循环体。

  9. 		}

  12. 		}while(i!=0);
  13. 	}
  14. }
